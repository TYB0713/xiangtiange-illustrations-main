向天歌 IP
角色定义
向天歌 / Xiang Tiange 是中文正文配图的固定视觉 IP。
默认每张图都要出现向天歌。向天歌不是站在角落里的讲解员，不是古风贴纸，也不是普通文人雅士，而是正在认真参与历史教学、文献考据、古籍鉴赏或知识传授的学究型伙伴。
外形
年轻学究型历史教师 / 古籍鉴赏从业者。
温润眉眼，专注有神，略带书卷气与考据者的严谨。
无框或细框眼镜，简洁斯文，偶尔推镜框的动作是标志性细节。
乌黑长发，束发或半束发，可用一根素色发带固定。
一枚青玉色发簪或发带结，是重要识别点。
素白长衫/白衬衫，清爽干净，袖口可微微卷起，露出执笔的手腕。
靛青色或墨蓝色外衫/马甲/背带，沉稳但干练，有旧书卷的气息。
白色布鞋或简约白鞋，可配素色袜。
体态自然，比例偏清爽手绘人物，不要夸张 Q 版。
线条干净，有手绘感；可以保留少量衣物纹理，但不要精致到商业立绘。
颜色
素白色：长衫/衬衫、背景、留白、纯净的行动感。
靛青/墨蓝：外衫/马甲、文献状态、补充说明、历史厚重感。
乌黑色：头发、线稿、主要文字。
浅肤色：人物皮肤。
青玉色：发簪/发带结、重点提示、考据标记。
赭石/暗红：只用于问题、提醒、错误、风险、待考疑点。
性格
细心、沉稳、好学不倦，有考据癖。
干练但亲和，像可靠的学究型伙伴。
偶尔有点书呆子气，说话会引经据典，但会认真把事情做完。
热爱历史与古籍，喜欢考据、茶道、碑帖、金石、阅读。
座右铭是"史笔如刀，一字不可轻；考据如鉴，一物不可妄"。
画面气质可以轻松、有幽默感，但不要卖萌，不要幼稚。
常见职责
让向天歌承担核心动作：
翻阅古籍、校勘文献、比对版本。
在书案前整理史料、梳理年表。
把混乱史料梳理成清晰脉络。
用镇纸压住失控的散页纸条。
用书本、卷轴、碑帖拓片搭出知识路径。
把输入素材分拣成文章、讲义、图示、清单。
给历史节点贴标签、连线、标记存疑处。
把复杂史实拆成几个可理解的小模块。
从一堆矛盾记载里挑出真正可靠的线索。
在旧书案、抽屉、信筒、书斋、看板等低科技物件旁认真操作。
拿着毛笔或钢笔修补断掉的知识链路或考证链条。
对着碑帖或器物认真鉴赏、记录批注。
给学生讲解历史脉络，在黑板上画出时间轴。
常用道具
古籍、线装书、碑帖拓片。
茶杯或茶壶。
毛笔、钢笔、砚台、镇纸。
便签、纸条、批注笺。
卷轴、年表、地图草图。
放大镜（用于鉴赏细节）。
猫咪元素可偶尔出现，但不要喧宾夺主。
禁止
不要画成小黑、黑色怪物、抽象黑豆角色。
不要画成普通古风美少年，必须保留学究/教师职业感。
不要过度飘逸、仙侠化、神话化。
不要画成商业海报主视觉、精致游戏立绘或偶像角色。
不要丢失细框眼镜、束发/半束发、青玉发簪/发带、素白长衫/衬衫、靛青外衫/马甲这些关键识别点。
不要让向天歌只是站在角落里看。
不要让向天歌抢走结构表达；他要推动结构成立。
不要加入复杂背景、宫殿大场景或真实 App 截图。
判断标准
如果去掉向天歌，图的核心隐喻还能完全成立，说明向天歌太装饰了；要重写提示词，让向天歌成为动作主体。
如果遮住脸和文字后，看不出"细框眼镜 + 束发/半束发 + 青玉发簪/发带 + 素白长衫/衬衫 + 靛青外衫/马甲"的组合，也说明 IP 识别度不够。
